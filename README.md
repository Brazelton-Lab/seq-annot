# seq-annot
Tools to fascilitate annotation and comparison of genomes and metagenomes

## About

seq-annot is a python package for annotating and counting genomic features
in genomes and metagenomes.

## Requirements

Python 3.4+

Python Libraries:

* arandomness
* bio_utils
* HTSeq

## Installation

pip install seq-annot
